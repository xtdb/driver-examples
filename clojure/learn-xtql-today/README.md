# Learn XTQL Today

An interactive Clerk notebook tutorial teaching XTQL, XTDB's data-oriented query language.

## Usage

Start a REPL:

```sh
clj -M:dev
```

Then evaluate the ns form in `src/learn-xtql-today-with-clojure.clj` and follow the instructions to start Clerk.

## Static Build

To generate a static HTML version:

```sh
./build-static.sh
```

Output will be in `public/build/`.
